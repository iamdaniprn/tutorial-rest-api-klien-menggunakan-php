# Tutorial Sederhana Rest API Klien Menggunakan PHP
 Projek ini menggunakan Bahasa pemrograman PHP dan Mysql sebagai databasenya.
 
 Koneksi database menggunakan PDO 
## Fitur:
	- Tampilkan Data Pasien
	- Tambah Data Pasien
	- Detail Data Pasien
	- Edit Data Pasien
	- Hapus Data Pasien
## Spesifikasi:
	- Support di PHP 7
